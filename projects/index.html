OpenWIS

OpenWIS is an implementation of WMO Information Systems (WIS) and aims to perform the tree functions required by the WIS, i.e. GISC, DCPC, and NC. It is originally developed by AKKA Information System (France) on behalf of the OpenWIS Association (with UKMO, MF, BoM, KMA, and MFI as founding members).

Excel2wis Tool

Excel2wis xml tool is a script to used to generate large set of XML metadata WMO Core 1.3 files from an excel file.

WIS2 Prototypes

The OpenWIS Association is conducting a set of WIS 2 prototype studies designed to test draft WIS2 concepts before they are implemented into a WIS2 version of OpenWIS.
